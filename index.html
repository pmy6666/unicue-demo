Abstract

This demo page organizes prompt-controlled target speech extraction into a single comparison matrix. Each row corresponds to one non-empty combination of four prompt types: visual, text, pre-enrollment, and spatial. The layout is designed to make listening comparisons direct and consistent across targets, while keeping the naming and presentation simple enough for paper reviewers and demo visitors.

Method Overview

A shared mixture is queried by four prompt channels and rendered as a fixed matrix of fifteen prompt combinations.
